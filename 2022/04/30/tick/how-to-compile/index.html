如何编译

git clone https://github.com/yarrick/iodine.git
由于没必要安装unix2dos所以建议注释掉Makefile中的unix2dos @unix2dos iodine-latest/*
安装mingw64和libz-mingw-w64-dev,libz-mingw-w64apt install mingw64 libz-mingw-w64-dev libz-mingw-w64
将作者给出的[zlib](https://code.kryo.se/iodine/deps/zlib.zip

) 中bin目录中的zlib1.dll文件放入/usr/x86_64-w64-mingw32/bin/zlib1.dll 和 /usr/i686-w64-mingw32/bin/zlib1.dll(也可以自己编译最新版，见此处)

cd iodine && make iodine-latest-windows.zip
