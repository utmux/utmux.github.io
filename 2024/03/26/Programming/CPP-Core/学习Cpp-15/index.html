类中函数的定义

类函数和普通函数的定义类似，区别如下

  1. 定义成员函数时，使用作用域解析运算符（::）来标识函数所属的类；
  2. 类方法可以访问类的 private 组件，比如私有函数等

故类函数无需传递或者定义即可访问类中的数据成员和函数成员。类函数和普通函数一样可以使用内联 inline，代表在每一个类中都复制一份代码，而不是调用该代码，例如：

根据 C++中的规则，inline 更像一种高级的复制，类似于 define 一样，不过更加智能。

批注：直接在类定义完整函数的类成员函数都是默认作为 inline 的。

为了节省空间，同一个类定义的不同变量，其数据成员的空间是分开的，而方法是共用同一个空间的，也就是调用同一个函数。

this 指针

C++中的每一个成员函数都有一个 this 指针，指向调用该成员函数的类，我们可以通过 *this 来返回或者访问这个类，也可以通过加入 const 来防止通过 this 指针修改或者调用类中的成员。
