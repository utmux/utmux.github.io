状态压缩 DP

所谓的状态压缩 DP，即借助二进制，将原本的状态进行压缩，压缩成 0 或 1，从而将原本需要的大量空间压缩到一个数字中，优化时间和空间复杂度。因而，状压 DP 通常涉及位运算，以下是几种常用的位运算。

用途 操作方式
将第 i 位上的数字修改为 1 `x = x
将第 i 位上的数字修改为 0 x = x & ~(1 << i - 1)
查询第 i 位上的数字是否为 1 x & (1 << i - 1) != 0

即通过另一种角度来看待这些状态，从而节省空间。

所谓的状态压缩，即比如如果要表示一个 3x3 的地图，此时我们可以使用位来表示，每个位代表 3x3 中每个点的情况。
