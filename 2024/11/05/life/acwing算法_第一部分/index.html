排序算法

下面这些算法大部分在 C/C++的标准库都进行了实现，比如：

  1. std::qsort（使用于数组，C 语言库的标准快速排序实现）：
    是 C 标准库中的快速排序函数，定义在 <cstdlib> 头文件中。
  2. std::bsearch：
    C 标准库中的二分查找函数，定义在 <cstdlib> 头文件中。std::bsearch(key, base, num, size, comp)，其中 key 是要查找的值，base 是指向数据的指针，num 是元素个数，size 是每个元素的大小，comp 是比较函数。
  3. std::sort（适用于 STL 容器数据）：
    采用了高级的排序算法（一般为快速排序或混合算法），其平均时间复杂度为 O(N*long(N))
  4. std::binary_search（适用于 STL 容器）：
    C++提供的二分查找，允许自定义比较函数，从而使用自定义的排序规则。
